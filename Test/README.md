# Test Directory

Эта директория содержит тестовые скрипты для проекта VLC.

## Правила тестирования

> **ВАЖНО:**
> - **ВСЕ тесты** выполняются на Raspberry Pi, НЕ на локальном компьютере
> - **Все тестовые файлы** создаются и сохраняются в этой директории
> - **НЕ удалять** тестовые файлы после выполнения
> - Тестовые файлы сохраняются для повторного использования и отладки

## Тестовые файлы

### `test-skip-wrappers.sh`
Тестирование bash обёрток для skip markers:
- `db_get_skip_markers()` - получение markers из БД
- `db_set_intro_markers()` - установка intro markers
- `db_set_outro_marker()` - установка outro marker
- `db_clear_skip_markers()` - очистка markers

**Использование на Raspberry Pi:**
```bash
cd /path/to/vlc
./Test/test-skip-wrappers.sh
```

## Добавление новых тестов

При создании нового тестового скрипта:
1. Создать файл в этой директории с именем `test-<feature>.sh`
2. Сделать его исполняемым: `chmod +x Test/test-<feature>.sh`
3. Добавить описание в этот README
4. Выполнить тест на Raspberry Pi
5. НЕ удалять файл после выполнения
