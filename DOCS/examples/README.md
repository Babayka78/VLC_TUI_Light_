# Examples / Примеры

Папка с примерами форматов файлов проекта.

## playback_format_example.txt

Пример формата файла `.playback` для отслеживания прогресса воспроизведения.

**Формат:**
```
filename:seconds:total:percent:timestamp
```

**Поля:**
- `filename` - имя видеофайла
- `seconds` - текущая позиция в секундах
- `total` - общая длительность в секундах
- `percent` - процент просмотра (0-100)
- `timestamp` - Unix timestamp последнего обновления

**Пример записи:**
```
video.avi:2673:2700:99:1732567890
```

Означает: файл video.avi просмотрен на 99% (2673 из 2700 секунд), последнее обновление 25.11.2024.

## Использование

Файлы `.playback` создаются автоматически скриптом `playback-tracker.sh` в той же директории, где находятся видеофайлы.
