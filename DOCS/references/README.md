# References / Справочники для разработки

Документация и справочники для работы с проектом.

## VLC RC Commands

Команды интерфейса VLC Remote Control (полный список в вашем `dev_notes/vlc_rc_commands.txt`).

### Ключевые команды для автосохранения:
- `get_time` - текущая позиция в секундах
- `get_length` - общая длина видео в секундах
- `is_playing` - проверка воспроизведения (1/0)

### Управление:
- `pause` - пауза/продолжить
- `seek X` - перемотка на X секунд
- `stop` - остановка
- `quit` - выход из VLC

## CEC Codes

Коды кнопок пульта для CEC управления.

### Основные кнопки:
```
44:00 - OK (Play/Pause)
44:0d - BACK (Exit)
44:35 - INFO (Show time)
```

### Навигация:
```
44:01 - UP (+30 sec)
44:02 - DOWN (-30 sec)
44:03 - LEFT (-10 sec)
44:04 - RIGHT (+10 sec)
44:30 - CH UP (+60 sec)
44:31 - CH DOWN (-60 sec)
```

### Цветные кнопки:
```
44:72 - RED (Audio track)
44:73 - GREEN (Subtitles)
44:74 - YELLOW (Volume +)
44:71 - BLUE (Volume -)
```

### Цифры (Jump to %):
```
44:20 - 0 (Start)
44:21-29 - 1-9 (10%-90%)
```

## Playback File Format

Формат файла `.playback` для отслеживания прогресса.

```
filename:seconds:total:percent:timestamp
```

**Пример:**
```
video.avi:2673:2700:99:1732567890
```

**Пороги:**
- `>= 99%` = `[X]` (просмотрено)
- `>= 1%` = `[T]` (частично)
- `0%` = `[ ]` (не просмотрено)
